

# Digital Mania — Premium SMM Marketplace

## Design System
- **Pure white (#FFFFFF) background** with heavy white space
- Electric Blue (#2563EB) primary, Royal Purple (#7C3AED) accent, Soft Gold (#D4AF37) luxury accent
- Inter font family, generous heading spacing
- Soft shadows, rounded cards, gradient buttons (blue→purple), smooth hover animations

## Pages

### 1. Landing Page
- **Hero**: Bold headline + subheadline, two CTA buttons, floating animated service cards with counting numbers
- **Platforms**: Luxury icon grid (Instagram, TikTok, YouTube, Twitter, Spotify, Facebook, Telegram)
- **Trust indicators**: Secure payments, fast delivery, trusted service, 24/7 support + animated metrics (50k+ orders, 10k+ customers, 4.9 rating)
- **How It Works**: 4-step visual flow
- **Testimonials**: Modern review cards
- **Popular Services**: Cards with price, delivery time, order button
- **Footer**: Links grid

### 2. Services Page
- Filter tabs by platform
- Service cards grid: name, price/1000, delivery speed, min/max, View Details + Order Now buttons

### 3. Service Details Page
- Product layout: description, price, speed, guarantee, refill policy
- Order form: link field, quantity selector, live price preview, order button

### 4. Dashboard
- Stats cards: Balance, Total/Pending/Completed orders
- Recent orders table with status badges
- New Order + Add Funds buttons

### 5. Order Page
- Fast 3-field form: service dropdown, link input, quantity input
- Live price calculation, submit button

### 6. Add Funds Page
- Payment method selection (Stripe, Crypto, PayPal cards)
- Balance display, amount input

### 7. Login / Register
- Minimal auth pages: email + password fields
- Toggle between Login and Create Account

## UX
- All data is mock/static (no backend)
- CSS animations for counters, card hovers, button transitions, page fades
- Fully responsive mobile-first design
- Navigation header with logo + links across all pages
